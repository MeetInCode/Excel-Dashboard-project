# Excel Dashboard Project

An end-to-end data analysis and visualization piece using Microsoft Excel. This repository contains the raw data, working project files, and images of the final dashboard and pivot tables used to analyze and report metrics.

## Contents
- **project.xlsx**: The main Excel workbook containing raw data, cleaned data, pivot tables, and the final interactive dashboard.
- **datasets/**: Directory containing the original datasets used for this project.
- **dashboard.png**: A preview image of the final Excel Dashboard.
- **pivot_tables.png**: A preview image of the pivot tables powering the dashboard.
